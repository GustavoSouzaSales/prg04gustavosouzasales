  ## UltraGás I Sistema de Gás

</div>

O UltraGás é uma aplicação web que simula um sistema de gás, permitindo que usuários visualizem informações sobre o fornecimento, acompanhem o consumo, solicitem serviços e gerenciem seus pedidos de forma prática.

Este projeto está sendo desenvolvido para a disciplina de Programação Web, com o objetivo de construir uma aplicação completa ao longo do semestre.


---



## 📂 Estrutura do Projeto


### infrastructure
Parte onde ficam os arquivos principais e recursos usados no projeto inteiro.

- **/assets**
  - **css/** -> Arquivos de estilização do site
  - **fonts/** -> Fontes utilizadas no projeto
  - **images/** -> Imagens e ícones (incluindo favicon)
  - **js/** -> Scripts JavaScript

- **/pages**
  - **about/** -> Página "Sobre"
  - **contacts/** -> Página de contato

- **index.html** -> Página inicial do projeto.
